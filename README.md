# AI Eyewear Assistant

ระบบต้นแบบสำหรับช่วยเลือกกำลังแว่นอ่านหนังสือเบื้องต้นด้วยแบบทดสอบ 5 ข้อ

## ความสามารถ

- ประเมินกำลังแว่นอ่านหนังสือเบื้องต้น
- เช็กอาการเสี่ยงที่ควรไปตรวจสายตาก่อน
- แสดงผล +0.50 ถึง +3.00
- แนะนำสินค้า demo ตามกำลังแว่น
- พร้อมอัปโหลดขึ้น GitHub Pages

## วิธีเปิดใช้งาน

เปิดไฟล์ `index.html` ด้วย Browser ได้ทันที

## วิธีแก้ลิงก์สินค้า

แก้ในไฟล์ `script.js` ตรงตัวแปร `products`

```js
link: "https://shopee.co.th/"
```

ให้เปลี่ยนเป็นลิงก์ Shopee Affiliate จริง

## โครงสร้างไฟล์

```text
ai-eyewear-assistant/
├── index.html
├── style.css
├── script.js
├── README.md
└── data/
    └── products.json
```

## หมายเหตุสำคัญ

ระบบนี้เป็นเพียงเครื่องมือประเมินเบื้องต้น ไม่ใช่เครื่องมือตรวจวัดสายตาทางการแพทย์
หากผู้ใช้มีอาการผิดปกติ ควรตรวจสายตากับผู้เชี่ยวชาญ
