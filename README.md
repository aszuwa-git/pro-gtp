# AI Eyewear Assistant V2

ระบบต้นแบบสำหรับช่วยเลือกกำลังแว่นอ่านหนังสือ และช่วยเลือกทรงแว่นตามรูปหน้า

## ความสามารถใน Version 2

- ประเมินกำลังแว่นอ่านหนังสือเบื้องต้น
- เช็กอาการเสี่ยงที่ควรไปตรวจสายตาก่อน
- อัปโหลดรูปหน้าตรง
- แสดง Preview รูป
- ให้ผู้ใช้เลือกรูปหน้าที่ใกล้เคียง
- แนะนำทรงแว่นที่เหมาะกับรูปหน้า
- จับคู่สินค้า demo จากกำลังแว่น + รูปหน้า

## วิธีเปิดใช้งาน

เปิดไฟล์ `index.html` ด้วย Browser ได้ทันที

## วิธีนำขึ้น GitHub Pages

1. อัปโหลดไฟล์ทั้งหมดขึ้น Repository
2. ไปที่ Settings
3. เลือก Pages
4. Source: Deploy from branch
5. Branch: main
6. Folder: /root
7. Save

## วิธีแก้ลิงก์สินค้า

แก้ในไฟล์ `script.js` ตรงตัวแปร `products`

```js
link: "https://shopee.co.th/"
```

ให้เปลี่ยนเป็น Shopee Affiliate Link จริง

## โครงสร้างไฟล์


## หมายเหตุสำคัญ

ระบบนี้เป็นเพียงเครื่องมือประเมินเบื้องต้น ไม่ใช่เครื่องมือตรวจวัดสายตาทางการแพทย์
การเลือกรูปหน้าเป็นการแนะนำด้านสไตล์ ไม่ใช่การวิเคราะห์ทางชีวมิติหรือการยืนยันตัวตน
